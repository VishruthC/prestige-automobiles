## ğŸ§ About the Project

**Prestige Dealership** is a full-stack web application designed to simulate the operations of a luxury car dealership. It serves two main purposes: providing customers with a seamless browsing experience and giving administrators powerful tools to manage inventory.

The goal of this project was to build a robust, scalable e-commerce platform that handles complex data filtering, user authentication, and responsive design.
